# 🕵️‍♂️ Gestor de Vagas & Otimizador de Currículos com IA

Uma API desenvolvida em Node.js projetada para automatizar a busca de vagas de emprego na área de Tecnologia da Informação (TI). O sistema realiza a raspagem de dados (*web scraping*) de portais de vagas em tempo real e, futuramente, utilizará Inteligência Artificial (LLM) para analisar a compatibilidade da vaga com o currículo do usuário, fornecendo feedbacks e sugestões de melhorias de forma personalizada.

---

## 🚀 Tecnologias Utilizadas

O projeto foi estruturado utilizando o ecossistema JavaScript/Node.js, priorizando performance e escalabilidade:

* **[Node.js](https://nodejs.org/):** Ambiente de execução do JavaScript no servidor.
* **[Express](https://expressjs.com/pt-br/):** Framework web rápido e minimalista para a criação das rotas da API.
* **[Puppeteer](https://pptr.dev/):** Ferramenta de automação que controla uma instância do Chromium em segundo plano, essencial para realizar *web scraping* em plataformas dinâmicas.
* **[Nodemon](https://nodemon.io/):** Ferramenta de desenvolvimento que reinicia o servidor automaticamente a cada alteração no código.
* **[Dotenv](https://github.com/motdotla/dotenv):** Gerenciador de variáveis de ambiente para garantir a segurança de dados sensíveis.
